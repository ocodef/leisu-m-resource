# leisu-m-resource

## 项目简介

`leisu-m-resource` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态资源文件的集中存放与版本管理用途。

仓库中的 HTML 页面均为独立构建，彼此之间无依赖关系，可用于快速预览、测试或作为前端资源存档。

## 目录结构

```
leisu-m-resource/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 公共资源文件（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- **pages/**：每个子目录或文件对应一个独立页面，命名清晰，便于识别。
- **assets/**：存放页面可能引用的公共静态资源，如图标、样式表或脚本文件。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面之间相互独立，不共享数据或状态。
- 归档内容可能包含示例性演示、工具页面或临时测试页面，不构成完整产品。
- 本仓库不保证所有页面长期可用或内容不变，仅作为资源存档使用。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务进行托管（需自行配置）。
